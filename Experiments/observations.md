# Traffic Observation Experiment

## Objective
Validate that network traffic is captured, decoded, and centrally analyzed by the monitoring pipeline.
 
## Method
- Generated standard network traffic.
- Observed protocol-level events generated by Suricata
- Verified ingestion and visualization in Kibana
